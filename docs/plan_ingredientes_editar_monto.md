# Plan de Trabajo: Editar Monto de Presentación

**Estado:** En desarrollo
**Rama:** feat/ingredientes-editar-monto

## 🎯 Objetivo
Permitir la edición directa del monto de una presentación de compra existente en la gestión de ingredientes.

## 📋 Tareas
- [ ] Habilitar la edición en el campo de monto de las presentaciones listadas.
- [ ] Sincronizar el cambio con el backend a través del endpoint correspondiente (actualizar presentación).
- [ ] Validar que el monto sea un número válido.
