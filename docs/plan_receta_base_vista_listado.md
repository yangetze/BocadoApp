# Plan de Trabajo: Vista de Listado de Recetas Base (3.1)

## 🎯 Objetivo
Implementar una vista previa que muestre el listado de recetas ya agregadas, con un estado inicial simplificado si no hay registros.

## 📋 Tareas
- [ ] Backend: Crear endpoint para listar todas las recetas base guardadas.
- [ ] Frontend: Desarrollar componente de listado de recetas.
- [ ] Frontend (Estado Vacío): Mostrar un botón central "Agrega tu primera receta base" si no existen datos.
- [ ] Frontend (Navegador): Al hacer clic en "Agregar", redirigir a la vista de creación con Drag & Drop actual.
