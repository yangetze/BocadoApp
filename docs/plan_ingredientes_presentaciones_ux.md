# Plan de Trabajo: Mejoras UX en Presentaciones de Ingredientes

**Estado:** En desarrollo
**Rama:** feat/ingredientes-presentaciones-ux

## 🎯 Objetivo
Mejorar la experiencia de usuario (UX) en la gestión de presentaciones de compra de ingredientes, solucionando problemas de visualización, edición y jerarquía de información.

## 📋 Tareas
- [x] **Boton Agregar (Desktop):** Analizar y corregir el botón de "Agregar presentación" en vista desktop que se corta. Evaluar (y aplicar) según los estándares UX si es mejor un botón flotante (FAB) o uno de ancho completo.
- [ ] **Edición de Monto:** Modificar la interfaz de presentaciones (modal o lista) para permitir la edición directa del monto/costo de una presentación existente.
- [ ] **Jerarquía de Información:** Cambiar el orden de visualización en la lista de presentaciones para que la **Marca** sea lo principal y luego se muestre el detalle de la **Presentación** (ej. cantidad/volumen).
- [ ] Validar que los cambios sean responsivos y mantengan la consistencia visual.
