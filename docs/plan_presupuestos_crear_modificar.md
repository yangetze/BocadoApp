# Plan de Trabajo: Crear y Modificar Presupuestos con Tests

**Estado:** En desarrollo
**Rama:** feat/presupuestos-crud-tests

## 🎯 Objetivo
Implementar la funcionalidad completa para crear y modificar presupuestos, asegurando su correcto funcionamiento mediante pruebas automatizadas (tests).

## 📋 Tareas
- [ ] **Backend:** Asegurar que los controladores y rutas para crear (`POST /budgets`) y modificar (`PUT /budgets/:id`) presupuestos estén completamente implementados y funcionales.
- [ ] **Frontend:** Implementar/conectar la interfaz de creación y modificación de presupuestos con el backend.
- [ ] **Pruebas (Tests):**
  - [ ] Escribir pruebas unitarias/integración en el backend para la creación de presupuestos.
  - [ ] Escribir pruebas unitarias/integración en el backend para la modificación de presupuestos.
  - [ ] Validar flujos de error (ej. validaciones de datos, presupuesto no encontrado).
